# 🎮 Python Small Games Collection

Welcome to the **Python Small Games Collection**! This repository is a curated space for simple, fun, and lightweight text-based and console games built entirely using Python. 

It is a perfect resource for beginners looking to understand core programming concepts like loops, conditionals, functions, and state management in Python.

---

## 🚀 Games Included

Here are the games currently available in this repository:

1. **🎨 Color Guessing Game**
   * A console-based game featuring a virtual wallet where you can deposit, withdraw, and wager balance to guess colors.
2. **🎲 More Games Coming Soon!**
   * (e.g., Number Guessing, Rock Paper Scissors, Tic-Tac-Toe, etc.)
